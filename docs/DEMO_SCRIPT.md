# 愈后食光 HealMeal 3 分钟演示脚本

## 00:00 - 00:30 建档与隐私信任

1. 打开首页，点击“去建档”。
2. 展示建档入口的隐私说明文案与同意勾选。
3. 打开《隐私说明》弹窗，强调：
   - 仅用于个性化建议
   - 不售卖、不投放广告
   - 可随时修改/删除
   - 家属联动可关闭
4. 完成基本字段并保存，进入首页。

## 00:30 - 01:00 首页陪伴感

1. 展示“今天的小记录”与“今日一件小事”。
2. 展示折叠式健康提示（只出现一次）。
3. 点击 AI 提问，演示骨架/流式反馈与友好状态。

## 01:00 - 01:45 餐单生成与多菜位

1. 进入“餐单”，展示“今天想怎么吃”区域。
2. 选择套餐模板（默认均衡版：2 菜 1 汤 + 主食建议）。
3. 输入/语音录入食材 chips。
4. 点击生成，展示菜位卡片（换一道 / 查看详情）。

## 01:45 - 02:20 打卡闭环

1. 进入“打卡”，展示饮食完成度按“标记完成”计算。
2. 演示运动：设置目标 -> 记录分钟 -> 标记完成 -> 达标提示。
3. 演示饮水：+200ml 快捷键 -> 记录饮水 -> 达标提示。
4. 选择情绪并写一句感受，保存后返回首页可见摘要更新。

## 02:20 - 02:50 生态社区真实互动（Supabase）

1. 进入“生态”，展示真实帖子列表（非本地假数据）。
2. 发布一条帖子（可带图）。
3. 对任意帖子点赞、收藏、评论，刷新后状态保持。
4. 打开“我的”，展示我发过/我点赞/我收藏。

## 02:50 - 03:00 家属便捷买菜跳转

1. 进入“家属”页，展示置顶“便捷买菜”模块。
2. 点击阿里入口（第一位）直接跳转第三方平台。
3. 说明其余入口（盒马/美团/京东到家/叮咚）均可实跳。
